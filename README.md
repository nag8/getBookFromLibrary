# getBookFromLibrary
 図書館の本を自動で予約する

## 仕様
- 大阪市図書館を想定

## 方法
1. 読書メーターの「[読みたい本](https://bookmeter.com/users/763253/books/wish)」から情報を[firebase](https://console.firebase.google.com/u/0/project/bookbook-eaf17/database/bookbook-eaf17/data)に転記
2. firebaseの未予約のものを予約
